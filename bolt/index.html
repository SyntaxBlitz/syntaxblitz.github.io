Bolt

Bolt is a grading assistant tool written for CS 1332 at Georgia Tech. It's not open-source at the moment (since I haven't made sure there are no secrets in the code that the TAs keep from students), but I have a demo video here. These are all sample (phony) submissions; I made them to introduce new TAs to grading. None of the code you see here is from a real student submission :)
